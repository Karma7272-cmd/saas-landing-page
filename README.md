# SaaS Landing Page

This is a modern and animated SaaS landing page built with Next.js, Tailwind CSS, and Framer Motion.

## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
